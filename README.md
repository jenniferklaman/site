# Jennifer Laman

Hi! I’m **Jennifer Laman**, a linguist and engineer. This site is my personal portfolio and blog, where I share my research, projects, and make an occational blog post.

---

## Featured Projects

- **[Mijikenda Online Dictionary](#)** – Collaborated with a native speaker to build a standardized picture dictionary using the Online Living Dictionary platform.  
- **[Data-Constrained ASR Model](#)** – Developed multilingual automatic speech recognition models for 20+ low-resource languages with HuggingFace and data augmentation.  
- **[FDIC Data Analysis Pipeline](#)** – Automated scraping and processing of 23+ years of FDIC financial reports into a SQL database for analysis.  
- **[Phoneme Perceptron](https://github.com/jenniferklaman/Perceptron.git)** – Neural network model to classify phonemes from speech recordings.  
- **[Markov's Muse](https://github.com/jenniferklaman/MarkovsMuse.git)** – Text generation tool using Markov chains for creative writing.
